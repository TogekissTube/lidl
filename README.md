# lidl
Linux Dynamic Linker for x86_64 (now is a parser thats read ELF header), in the future will be mpre
Created for education and fun
It is:
400% more faster than glibc parsing files
100% x86_64
Minimal Codebase
Auditable
ASM optimized parser

WARNING: This DL is experimental, it has critical vulnerabilities and fails (try executing splint in *.c and *. or valgrind in the test executable), If something happens to your PC, its not our responsability, THIS IS NOT PRODUCTION-READY and if question, No, this is not a replace of Glibc, this is not functional and try to replace it will mess up your system, this is for experimenting
